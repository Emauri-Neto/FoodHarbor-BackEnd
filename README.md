**Back-end:** Django Restframework

## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/Emauri-Neto/FoodHarbor-BackEnd.git
```

Instale as dependências

```bash
  pip install -r requirements.txt
```

Inicie o servidor

```bash
  python manage.py runserver
```
